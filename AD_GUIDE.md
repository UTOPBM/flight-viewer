# 광고 이미지 가이드

## 📐 이미지 규격

### 배너 광고 (상단/하단)

#### 표시 크기
- **모바일** (375px 이하): 화면 너비 × 70px (비율 유지)
- **태블릿** (768px~1023px): 화면 너비 × 96px
- **데스크톱** (1024px 이상): 화면 너비 × 112px

#### 권장 이미지 사이즈
- **권장**: 2560 × 224px (레티나 디스플레이 대응)
- **최소**: 1280 × 112px
- **비율**: 약 11:1 (와이드 배너 형태)

#### 파일 형식 및 용량
- **형식**: JPG, PNG, WebP
- **최대 용량**: 500KB 이하 권장
- **해상도**: 72dpi 이상, 144dpi 권장

---

## 🎨 디자인 가이드라인

### 안전 영역
- 좌우 끝 **30px**는 텍스트/로고 배치 지양
- 중요 정보는 이미지 **중앙 70%** 영역에 배치

### 가독성
- **명도 50% 이상** 권장 (다크모드 대응)
- 텍스트 사용 시 **16px 이상** 크기
- 배경과 텍스트 **명도 차이 충분히** 확보

### 다크모드 고려
- 밝은 배경 이미지 권장
- 어두운 이미지는 테두리/그림자 추가 고려

광고 이미지 제출 전 확인사항:

- [ ] 이미지 크기: 최소 1280×112px, 권장 2560×224px
- [ ] 파일 용량: 500KB 이하
- [ ] 파일 형식: JPG, PNG, WebP 중 하나
- [ ] 중요 정보가 중앙 영역에 배치되었는가
- [ ] 텍스트 크기가 16px 이상인가
- [ ] 밝은 배경으로 다크모드 대응 가능한가
