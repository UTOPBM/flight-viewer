# 광고 이미지 가이드

## 📐 이미지 규격

### 배너 광고 (상단/하단)

#### 표시 크기 (고정 높이)
광고는 **고정 높이**로 표시되며, 너비는 화면 전체를 차지합니다:

- **모바일** (~767px): 화면 전체 너비 × **70px 고정**
- **태블릿** (768px~1023px): 화면 전체 너비 × **96px 고정**
- **데스크톱** (1024px~): 화면 전체 너비 × **112px 고정**

#### 권장 이미지 사이즈
고정 높이에 맞춰 이미지가 크롭(잘림)되므로, **가로로 여유있게** 제작하세요:

- **권장**: **2560 × 224px** (레티나 디스플레이 대응, 2배 해상도)
- **최소**: **1920 × 140px** (Full HD 기준)
- **안전**: **2000 × 200px** (중간 사이즈, 대부분 화면 대응)

> **중요**: 이미지는 높이에 맞춰 크롭됩니다 (`object-fit: cover`). 중요한 내용은 이미지 **중앙**에 배치하세요!

#### 파일 형식 및 용량
- **형식**: JPG, PNG, WebP
- **최대 용량**: **300KB 이하** 권장 (빠른 로딩)
- **해상도**: 72dpi 이상, **144dpi 권장** (레티나 대응)

## 🎨 디자인 가이드라인

### 안전 영역 (매우 중요!)
이미지가 크롭되므로 안전 영역을 준수하세요:

- **좌우 끝 50px**: 텍스트/로고 배치 **절대 금지** (잘릴 수 있음)
- **상하 끝 10px**: 여백 확보
- **중요 정보**: 이미지 **중앙 60~70%** 영역에만 배치

```
[← 50px 여백 →] [핵심 콘텐츠 영역] [← 50px 여백 →]
```

### 가독성
- **배경 명도**: 50% 이상 권장 (다크모드 대응)
- **텍스트 크기**: 최소 18px, 권장 24px 이상
- **폰트 굵기**: Bold (700) 이상 권장
- **대비**: 배경과 텍스트 명도 차이 충분히 확보 (WCAG AA 이상)

### 다크모드 고려
사이트는 다크모드를 지원합니다:

- ✅ **밝은 배경 이미지** 권장 (양쪽 모드에서 잘 보임)
- ✅ **텍스트에 윤곽선/그림자** 추가 시 가독성 향상
- ❌ 어두운 배경 이미지는 다크모드에서 보이지 않을 수 있음

## 📱 반응형 동작

광고는 화면 크기에 따라 **고정 높이**로 표시됩니다:

### 모바일 (~ 767px)
```
화면 너비: 375px ~ 767px (가변)
광고 높이: 70px 고정
표시 예시: 375×70px, 414×70px, 640×70px
```

### 태블릿 (768px ~ 1023px)
```
화면 너비: 768px ~ 1023px (가변)
광고 높이: 96px 고정
표시 예시: 768×96px, 820×96px
```

### 데스크톱 (1024px ~)
```
화면 너비: 1024px ~ 무제한 (가변)
광고 높이: 112px 고정
표시 예시: 1280×112px, 1920×112px, 2560×112px
```

> **참고**: 너비가 넓은 화면에서는 이미지 좌우가 크롭되므로, **2560px 이상 너비**로 제작 권장

## ✅ 체크리스트

광고 이미지 제출 전 확인사항:

- [ ] **이미지 크기**: 최소 1920×140px, 권장 **2560×224px**
- [ ] **파일 용량**: 300KB 이하
- [ ] **파일 형식**: JPG, PNG, WebP 중 하나
- [ ] **안전 영역**: 좌우 끝 50px, 상하 끝 10px 여백 확보
- [ ] **중요 정보**: 이미지 중앙 60~70% 영역에만 배치
- [ ] **텍스트 크기**: 최소 18px, 권장 24px 이상
- [ ] **폰트 굵기**: Bold (700) 이상
- [ ] **다크모드 대응**: 밝은 배경 또는 텍스트 윤곽선 있음
- [ ] **고해상도**: 144dpi (레티나 디스플레이 대응)

## 📌 FAQ

**Q. 광고 이미지가 잘리는데요?**
A. 고정 높이 시스템이라 좌우가 크롭됩니다. 2560px 이상 너비로 제작하고, 중요 정보는 중앙에 배치하세요.

**Q. 다크모드에서 광고가 안 보여요.**
A. 밝은 배경 이미지를 사용하거나, 텍스트에 윤곽선/그림자를 추가하세요.

**Q. 광고 효과를 측정할 수 있나요?**
A. 네, 노출 수와 클릭 수를 실시간으로 확인 가능합니다.
